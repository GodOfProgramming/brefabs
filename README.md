# Brefabs (Bevy Prefabs)

This crate provides a simple plugin to provide prefabs to Bevy.

# Usage

Define either a `Prefab` or a `StaticPrefab` and register it with the Plugin.

See the [simple](examples/simple.rs) example for basic usage.

# Versions
